 Structure
--------------

[robocode]          - Core
[robocodeui]        - Java implementation of UI
[robocodeextract]   - files to deploy during installation
[roborumble]        - competition runner
[codesize]          - utility to measure size of code
[installapplet]     - code of installer
[tools]             - various tools for developement
[nrobocode]         - .NET base library used by robots
[nrobocodeui]       - .NET implementation of UI
[nsamples]          - sample robost for .NET

Basic Compilation How To 
-------------------------------
1) you will need:
 - Java SDK
 - Ant
2) you may like to use
 - Eclipse
 - Jacobe 
 - Subversion
3)
 - get whole tree
 - cd build
 - ant build
